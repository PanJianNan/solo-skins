每个皮肤单独存放于根目录下以该皮肤名称为名的文件夹中。
其中 skin-preview 存放皮肤的预览图片，toolers 存放更新版本和压缩 js、css 的 NodeJS 脚本，部署时不要拷贝这两个目录哦~
